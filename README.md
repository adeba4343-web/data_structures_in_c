# data_structures_in_c
A growing collection of Data Structures and Algorithms implemented in C for building strong programming fundamentals.
This repository demonstrates data structures like:
-binary tree
-binary search tree
-AVL tree
-linked list
  -singly linked list
  -doubly linked list
  -circular linked list
-stack using arrays
-queue using arrays
  -linear queue
  -circular queue
-sorting techniques like
  -merge sort
  -selection sort
  -quick sort
  -bubble sort
-searching texhniques like
  -binary search
  -linear search
-infix to postfix expression
